The following KPIs are used to evaluate portfolio quality, monitor credit risk, and support business interpretation.

| KPI                       | Definition                            | Business Purpose                 |
| ------------------------- | ------------------------------------- | -------------------------------- |
| Portfolio Size            | Number of loans included in analysis  | Measure portfolio scale          |
| Total Exposure            | Total approved loan amount            | Measure lending volume           |
| Average Loan Amount       | Average loan amount per borrower      | Understand loan distribution     |
| Default Rate              | Percentage of defaulted loans         | Measure portfolio credit quality |
| Default Count             | Number of defaulted loans             | Monitor risk concentration       |
| Grade Default Rate        | Default rate by borrower grade        | Compare risk across segments     |
| DTI Distribution          | Distribution of debt-to-income ratio  | Evaluate borrower leverage       |
| Loan Purpose Distribution | Distribution of borrowing purposes    | Understand customer behavior     |
| State-level Default Rate  | Default rate by geographic region     | Detect regional concentration    |
| Vintage Performance       | Portfolio performance by issue period | Monitor risk trends over time    |
| KPI                       | Formula                   | Purpose                 |
| ------------------------- | ------------------------- | ----------------------- |
| Approval Volume           | Count approved loans      | Growth                  |
| Active Loan Ratio         | Active / Total            | Portfolio health        |
| Delinquency Rate          | Late loans / Active loans | Early warning           |
| Charge-off Rate           | Charged off / Total       | Loss                    |
| Avg Interest Rate         | AVG(int_rate)             | Yield                   |
| Vintage Default Rate      | Default by issue cohort   | Monitoring              |
| Exposure at Default (EAD) | Outstanding balance       | Risk exposure           |
| Loss Rate                 | Loss / Exposure           | Portfolio profitability |
| Delinquency Rate          | Late Loans / Active Loans | Monitor early-stage repayment deterioration |
